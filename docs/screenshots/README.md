# Screenshots

Put Foundry UI screenshots and other images in this folder.

Conventions:

- Prefer PNG for UI screenshots.
- Use short, descriptive names: `<topic>-<yyyy-mm-dd>.png`
- Keep files reasonably sized (crop to the relevant UI).

To reference an image from `README.md`:

```md
![Compute module run in Foundry](docs/screenshots/compute-module-run-2026-02-15.png)
```
